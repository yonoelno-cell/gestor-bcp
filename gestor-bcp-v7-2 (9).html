OpenAI GPT-4o — La más recomendada ⭐
Extrae monto, fecha, N° operación y banco directamente en formato estructurado, sin necesidad de texto intermedio. Precisión superior en comprobantes peruanos.

Obtener clave: platform.openai.com → API Keys → Create new secret key
Costo: ~$0.004 por imagen (muy económico)
OpenAI API Key
Google Gemini Flash — Gratuita 💎
Muy buena precisión y totalmente gratis hasta 1,500 solicitudes/día. Ideal si quieres cero costos.

Obtener clave: aistudio.google.com → Get API Key → Create API Key
Costo: Gratis (Gemini 2.0 Flash)
Google Gemini API Key
Google Cloud Vision — OCR clásico 👁
Extrae texto puro del comprobante. Buena precisión, 1,000 imágenes/mes gratis.

Obtener clave: console.cloud.google.com → APIs → Cloud Vision API → Credenciales → Crear clave de API
Costo: Gratis (1,000/mes), luego $1.50 por 1,000
Google Cloud Vision API Key
Tesseract.js — Motor local 📦
Funciona sin internet ni clave. Menor precisión en comprobantes con fondos complejos. Recomendamos cambiar a GPT-4o o Gemini para mejores resultados.

Recomendación para tu base de datos de 1.4 GB

🥇 Supabase (PostgreSQL) — La mejor opción
· Plan gratuito: 500 MB (ampliable en planes pagados ~$25/mes)
· Tiene API REST automática: el HTML puede consultar directamente
· Soporta millones de registros DNI con búsqueda instantánea
· Cómo migrar: extrae el RAR → importa CSV a Supabase

🥈 PocketBase — Autoalojado, gratis
· Instalas en tu PC/VPS, funciona con archivos SQLite grandes
· Sin límites de almacenamiento, muy fácil de usar

🥉 Turso — SQLite distribuido
· 9 GB gratis, ultra rápido para lecturas, API HTTP directa
